| Command                         | Description                                           |
| ------------------------------- | ----------------------------------------------------- |
| ls                              | List files in directory                               |
| cd `<directory>`                | Change directory `cd ../` to go up, `cd ~` to go home |
| pwd                             | Display path of the current directory                 |
| mkdir `<directory>`             | Make directory                                        |
| touch `<file>`                  | Create file                                           |
| rm -r `<directory>`             | Delete directory                                      |
| rm `<file>`                     | Delete File                                           |
| nano `<file>`                   | Edit file                                             |
| mv `<directory><file>` location | Move a file or directory to a new location            |
| cat                             | Outputs the content of a file                         |
| echo                            | Write contents onto a file                            |
| >				  | Overwrites a file					  |
| >>				  | Appends an already present file or creates the file	  |
